Quick start

This page should help you start work, not sell you the app. These are the primary moves inside Pixelbox.

1. Choose a project

Use the menu to switch into a project folder. Pixelbox keeps preview state and terminal context scoped to that project.

2. Open the terminal

Resume Codex or run commands directly. The terminal is the control surface for booting previews and doing real work.

3. Point the preview

Set a local URL or HTML file in Running Page so the result stays visible while you iterate.
