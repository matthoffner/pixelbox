# matthoffner

A design-forward developer landing page with a zero-dependency local dev server.

## Run locally

```bash
npm run dev
```

Then open:

```text
http://127.0.0.1:3000
```

Project pages:

```text
http://127.0.0.1:3000/pixelbox.html
http://127.0.0.1:3000/automoat.html
http://127.0.0.1:3000/forge.html
```
