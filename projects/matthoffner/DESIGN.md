# Matthoffner Design System

## Direction

Build this as a cinematic developer landing page, not a generic SaaS site. The tone should feel like a design-forward creative studio crossed with an elite engineering portfolio: sharp, editorial, immersive, and technically confident.

## Visual Principles

- Full-bleed composition with large atmospheric surfaces instead of boxed cards everywhere.
- Typography carries the hierarchy. Headlines should feel poster-like and intentional.
- Use warm neutrals, mineral blues, oxidized teals, and a restrained electric accent instead of default black/white/purple.
- Mix precision and texture: grid lines, soft radial glows, glass panels, and dense microcopy.
- Motion should feel structural. Sections reveal with depth and offset, not generic bounce effects.

## Type

- Headlines: `"Georgia", "Times New Roman", serif`
- Interface and body: `"Helvetica Neue", "Arial", sans-serif`
- Labels and data: `"Courier New", monospace`

The serif display type gives the site a more authored identity than standard tech-product typography.

## Layout

- Hero should occupy the first viewport and feel immediate.
- Use asymmetry: offset columns, overlapping panels, and broken grid lines.
- Sections should alternate between editorial storytelling and technical proof.
- Keep mobile responsive by collapsing to a single column and reducing decorative overflow.

## Motion

- Slow ambient gradients in the background.
- Staggered entrance reveals using opacity, blur, and translate.
- Hover interactions should slightly tilt or lift key panels.

## Content Strategy

- Lead with a point of view, not a resume.
- Show selected capabilities as crafted systems, not bullet-heavy skill dumps.
- Make case-study teasers feel like invitations into deeper work.
